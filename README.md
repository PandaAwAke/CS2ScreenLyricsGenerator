# 说明
根据 lrc 和字体自动生成中英文歌词字幕粒子和 Trigger，如果歌词有重复语句需要消除得人工操作。

## 使用方法
1. 在 lyrics 放进要生成的 lrc 歌词，自己调句子。目前仅支持中英文，纯英文不知道什么效果。
2. 点进 main.py 自己改配置
3. 运行 main.py

## 输出
在 output 文件夹下输出：材质、粒子、Trigger 文本。

材质和粒子直接复制到你的项目 content 就能用。

Trigger 文本绑定了一个特定名字的 relay，你需要在 Hammer 里面自己创建对应的 relay 然后点开 Output Paste 即可。注意这里的 Trigger 第一句歌词是从 0 秒开始的，后面都是相对时间。你可以自己调什么时候触发 relay。

## 自定义模板

在 templates 里面有 3 个自定义模板，要改的话自己弄一个，然后把内容替换进去，在需要替换的地方改成三个井号 ### 即可。
